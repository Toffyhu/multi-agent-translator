# 专业翻译技能

## 格式保留
- 不翻译代码块、HTML标签、Markdown语法
- 保留占位符 `{name}` `%s` 保持原样
- 保留文档结构：标题、链接、粗斜体
- 保留JSON/XML键名，只翻译值

## 内容规则
- 不翻译：专有名词、品牌名、技术术语、URL、邮箱
- 不翻译：代码片段、CSS类名、API端点、文件扩展名
- 数字和ID保持原格式，除非需要本地化转换
- 全文术语保持一致

## 中英语言适配
- 英语前重心 → 中文后重心
- 英语树状嵌套 → 中文竹节铺排
- 被动语态 → 主动语态
- 物称主语 → 人称主语

## 文化适配
- 习语本土化而非直译
- 单位换算（英里↔公里、华氏↔摄氏）
- 日期格式按目标地区调整
- 货币符号和数字格式本地化

## 质量控制
- 先读全文把握含义
- 检查译文是否自然流畅
- 验证所有格式元素完好
- 确保全文术语和语气一致
